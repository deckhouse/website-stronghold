Добро пожаловать в документацию **Deckhouse Stronghold**.

Deckhouse Stronghold — продукт для безопасного хранения секретов, управления доступом к ним и автоматизации жизненного цикла конфиденциальных данных. Решение поддерживает сценарии хранения статических секретов, выпуска динамических учётных данных, шифрования данных и интеграции с внешними системами аутентификации и управления ключами. API продукта совместим с **HashiCorp Vault API**.

Документация поможет:
- познакомиться с возможностями продукта и его архитектурой;
- выбрать подходящий вариант развёртывания;
- установить и настроить Deckhouse Stronghold;
- организовать безопасную эксплуатацию;
- начать работу с аутентификацией, политиками и механизмами секретов;
- использовать справочные материалы по API, CLI и параметрам конфигурации.

## Разделы документации

- [О продукте](/products/stronghold/documentation/О продукте/) — назначение Deckhouse Stronghold, ключевые возможности, архитектура, редакции, системные требования, совместимость и история версий.
- [Руководство по развертыванию](/products/stronghold/documentation/Руководство по развертыванию/) — варианты развёртывания, установка в Standalone и в DKP, первичная настройка, проверка работоспособности и удаление.
- [Руководство администратора](/products/stronghold/documentation/Руководство администратора/) — эксплуатация, управление доступом, аудит, резервное копирование, восстановление, безопасность, обновление и диагностика.
- [Руководство пользователя](/products/stronghold/documentation/Руководство пользователя/) — начало работы, основные понятия, методы аутентификации, управляемые ключи, механизмы секретов и Stronghold Agent.
- [Сценарии использования](/products/stronghold/documentation/Сценарии использования/) — практические примеры работы с OIDC, AppRole, KV, Transit, PKI, Kubernetes и внешними KMS/HSM.
- [Справочник](/products/stronghold/documentation/Справочник/) — API, CLI и консольные утилиты, параметры конфигурации, глоссарий, FAQ и коды ошибок.

## С чего начать

Если вы только знакомитесь с продуктом, рекомендуем следующий порядок:
1. Перейдите в раздел [О продукте](/products/stronghold/documentation/О продукте/), чтобы понять возможности и ограничения Deckhouse Stronghold.
2. Откройте [Руководство по развертыванию](/products/stronghold/documentation/Руководство по развертыванию/) и выберите подходящий вариант установки.
3. После установки используйте [Руководство администратора](/products/stronghold/documentation/Руководство администратора/) для базовой настройки и безопасной эксплуатации.
4. Для повседневной работы с секретами, токенами и методами аутентификации используйте [Руководство пользователя](/products/stronghold/documentation/Руководство пользователя/).
